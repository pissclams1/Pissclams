# GapStay

GapStay helps Airbnb and VRBO hosts turn vacant 14-90 day calendar gaps into realistic mid-term furnished-stay offers.

## What this MVP includes

- Host-facing landing page
- Gap analyzer for vacant dates
- Dynamic mid-term pricing recommendation
- Generated public listing pages
- Guest inquiry capture
- Host dashboard using local browser storage
- Vercel-ready Next.js app

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Import this repository/branch into Vercel as a Next.js project. No environment variables are required for the demo MVP.

## Product thesis

Hosts over-anchor on fantasy nightly rates. Guests want honest furnished stays without Airbnb fee shock. GapStay packages empty calendar gaps into realistic, all-in offers that are easier for both sides to say yes to.
